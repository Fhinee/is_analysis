## 4.接口说明
- 接口名：delStudent
- 功能：删除学生及先关信息
    
- 权限： 老师
    
- API请求地址：  接口基本地址/v1/api/delStudent

- 请求方式 ：POST

- 请求实例：
```
 {
      "student_id":1234,
  }
```     

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |student_id|学生学号|

    
- 返回实例：
```
  {         
      "status": true,
      "info": ok
  }
```
- 返回参数说明：

 |参数名称|说明|
 |:---------:|:--------------------------------------------------------|
 |status|bool类型，true表示正确的返回，false表示有错误|
 |info|ok说明删除成功|
