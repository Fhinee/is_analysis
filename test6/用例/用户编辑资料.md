## 4.接口说明
- 接口名：setInformation
- 功能：编辑用户的用户信息、密码
    
- 权限： 学生/老师：编辑自己资料
    
- API请求地址：  接口基本地址/v1/api/setInformation

- 请求方式 ：POST

- 请求实例：
```
 {
      "user_id":1234,
      "github_username":Fhinee
      "password":AF#W@#AAAASDF
  }
```     

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |user_id|用户编号|
  |github_username|github用户名|
  |password|用户密码|
    
- 返回实例：
```
  {         
      "status": true,
      "info": null
  }
```
- 返回参数说明：

 |参数名称|说明|
 |:---------:|:--------------------------------------------------------|
 |status|bool类型，true表示正确的返回，false表示有错误|
 |info|返回结果说明信息|
