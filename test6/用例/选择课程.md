## 4.接口说明
- 接口名：chooseCourse
- 权限：老师、学生
- API请求地址： 接口基本地址/v1/api/chooseCourse
- 请求方式 ： POST
- 请求实例
```
  {         
      "course_id": 20,
      "course_name": 软件分析,
      "update_time": 2018-8-5
  }
```
 - 请求参数说明：
 
 |参数名称|说明|
    |:---------:|:-------------------------------------------|
    |course_id|课程编号|
    |course_name|课程名称 |
    |update_time|操作日期|

- 返回实例
```
  {         
      "status": true,
      "info": null
  }
```
- 返回参数说明：

 |参数名称|说明|
 |:---------:|:--------------------------------------------------------|
 |status|bool类型，true表示正确的返回，false表示有错误|
 |info|返回结果说明信息|
